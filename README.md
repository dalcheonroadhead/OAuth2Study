# 🧸 Spring Play Ground 🧸

#### ✏️ 프로젝트 단위로 실습하고, 디버깅 하고, 정리합니다. 



### 1. 학습 방법 

A. 이해하고 싶은 개념을 정해서, ``ISSUE TEMPLATE``에 맞게 작성합니다. 

B. 해당 파트 관련 프로젝트를 ``실습``합니다.

C. 이해하고 싶은 내용을 ``강의하는 스크립트``를 작성하여 블로그에 올립니다. 

​    (최대한 스스로 작성해보고, 모르는 것은 다시 학습합니다.)
